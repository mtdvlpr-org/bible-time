# Changelog

## 1.0.0 (2025-11-18)

### Features

* add changelog viewer to see recent changes in the app.
* add contact form for users to reach out to the team.
* add detail pages for Biblical people and events.
* add possibility to add, edit, and delete people and events for admins.
* add possibility to make suggestions for new people and events or edits to existing ones for regular users.
* add possibility to review suggestions for moderators.
* add possibility to translate people and events for translators.
* add settings to customize the app experience.
* add support for multiple languages.
* add table overviews of Biblical people and events with search, filters, and sorting.
* add timeline view for Biblical people and events.
* allow anyone to sign up and create an account.
